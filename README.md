# aMALgamous



**##Install##**
```
git clone https://github.com/Marantral/aMALgamous.git
cd aMALgamous 
sudo ./setup.sh 
```
**##Run/Operate##**
```
python3 aMALgamous.py
```

**All malware will be placed in aMALgamatiom/current with old malware being placed in an arcive folder.**
